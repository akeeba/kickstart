Release highlights

Updated design. You asked for flatter design, you get a flatter design.

Option to not rename the .htaccess file before extraction. Use with caution, it might break the extraction. Useful for specific cases where you want to keep your .htaccess file because it's required to enable the correct PHP version on your site.

PHP 5.3.04+ is required

Akeeba Kickstart requires PHP 5.3.04+, 5.4, 5.5, 5.6 or 7.0. It won't run on hosts running PHP 5.3.03 (usually written as 5.3.3) or earlier, including PHP 5.2, 5.1, 5.0 and all 4.x versions.
