Release highlights

Improved PHP 7.1 support. Now using OpenSSL instead of mcrypt whenever possible to handle encrypted (JPS) archives and other cryptographic needs.

Bug fixes. Known issues with the Import from URL have been addressed in this release.

PHP 5.3.04+ is required

Akeeba Kickstart requires PHP 5.3.04+, 5.4, 5.5, 5.6 or 7.0. It won't run on hosts running PHP 5.3.03 (usually written as 5.3.3) or earlier, including PHP 5.2, 5.1, 5.0 and all 4.x versions.
