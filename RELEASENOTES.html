Release highlights

New theme. Kickstart was long overdue for a new lick of paint. We changed our previous theme to a more flat design with the same color scheme we use in all of our other software.

Dark Mode. Embrace the dark side! When your browser is signalling that you're using a dark theme on a compatible operating system (macOS, Windows 10, Linux / GTK3, iOS, Android) Kickstart will automatically change its theme to vibrant color on a dark background. If you're a night own like us your eyes will thank you for the switch.

PHP 5.3.04+ is required

Akeeba Kickstart requires PHP 5.3.04+, 5.4, 5.5, 5.6, 7.0, 7.1, 7.2, 7.3 and 7.4. It won't run on hosts running PHP 5.3.03 (usually written as 5.3.3) or earlier, including PHP 5.2, 5.1, 5.0 and all 4.x versions.
